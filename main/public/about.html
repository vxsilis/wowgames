About
Thank you for visiting Wow Games! Here, you can invite friends and play classic games online!

This project is part of the undergraduate thesis of Vasilis Gavriilidis, based on node.js and socket.io. The source code and the assets are free and available on Github. You can watch the video presentation here. Because of the server's low CPU power not all games are available at all times.

Player's options like name, type of gameplay and game they want to play are stored in a database. They are used only for the game to function properly. No marketing or statistic reasons.
